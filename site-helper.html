<HTML>

HTML — язык гипертекстовой разметки сайта, помогающий структурировать содержание каждой веб-страницы. С помощью HTML разработчик собирает «скелет» сайта.

a
<a href=""></a>
Парный тег, определяет гиперссылку, которая используется для перехода с одной страницы на другую.
abbr
<abbr title=""></abbr>
Парный тег, определяет сокращение, например "HTML", "CSS".
acronym
<acronym title="></acronym>
Парный тег, позволяет авторам четко указать последовательность символов, составляющих аббревиатуру слова. Тег не рекомендуется, лучше используйте <abbr>.
base
<base href=""/>
Одиночный тег, задает базовый URL, который будет использоваться для всех относительных URL-адресов в документе. В документе может быть только один <base> элемент.
basefont
<basefont />
Одиночный тег, предназначен для задания шрифта, размера и цвета текста по умолчанию. Указанные значения будут использоваться во всем документе за исключением тега <font>.
br
<br />
Одиночный тег, создает перевод строки в тексте.
hr
<hr />
Одиночный тег, Рисует горизонтальную прямую.
bdo
<bdo dir=""></bdo>
Парный тег, переопределяет текущую направленность текста, текст внутри отображается в другом направлении.
colgroup
<colgroup></colgroup>
Парный тег, определяет группу столбцов в таблице.
col
<col />
Парный тег, определяет один или несколько столбцов в группе столбцов, представленной его родительским <colgroup> элементом.
link
<link rel="stylesheet" href="" />
Одиночный тег, чаще всего используется для создания ссылок на таблицы стилей, но также используется для создания значков сайта.
meta
<meta />
Одиночный тег, предоставляет метаданные или информацию о HTML-документе. Он включает в себя такую информацию, как набор символов, описание, ключевые слова и другие важные детали.
style
<style></style>
Парный тег, содержит информацию о стиле документа или части документа.
script
<script></script>
Парный тег, используется для встраивания исполняемого кода или данных; обычно он используется для встраивания кода JavaScript или ссылки на него.
img
<img src="" alt=""/>
Одиночный тег, встраивает изображение.
source
<source />
Одиночный тег, обычно используется для предоставления одного и того же медиаконтента в нескольких форматах файлов, чтобы обеспечить совместимость с широким спектром браузеров.
iframe
<iframe src="" frameborder="0"></iframe>
Парный тег, встраивает другую HTML-страницу в текущую.
embed
<embed src="" type="" />
Одиночный тег, вставляет расширенный контент в выбранное место документа.
object
<object data="" type=""></object>
Парный тег, представляет внешний ресурс, который может обрабатываться как изображение, вложенный контекст просмотра или ресурс, обрабатываемый плагином.
area
<area shape="" coords="" href="" target="" alt=""/>
Парный тег, определяет область внутри карты изображения, которая имеет предопределенные области с возможностью кликабельности.
map
<map name=""></map>
Парный тег, используется с <area> элементами для определения карты изображений (области кликабельных ссылок).
form
<form action=""></form>
Парный тег, представляет раздел документа, содержащий интерактивные элементы управления для отправки информации.
label
<label for=""></label>
Парный тег, представляет заголовок для элемента в пользовательском интерфейсе.
input
<input type="" id=""/>
Одиночный тег, используется для создания интерактивных элементов управления для веб-форм с целью приема данных от пользователя.
select
<select name="" id=""></select>
Парный тег, представляет элемент управления, который предоставляет меню опций.
textarea
<textarea name="" id="" cols="30" rows="10"></textarea>
Парный тег, представляет собой многострочный элемент управления редактированием обычного текста, полезный, когда вы хотите разрешить пользователям вводить значительный объем текста произвольной формы.
video
<video src=""></video>
Парный тег, встраивает в документ медиаплеер, поддерживающий воспроизведение видео.
audio
<audio src=""></audio>
Парный тег, используется для встраивания звукового контента в документы.
keygen
<keygen />
Одиночный тег, используется для указания поля генератора пары ключей в форме.
command
<command />
Одиночный тег, создает команду в виде переключателя, флажка или обычной кнопки.
bq
<blockquote></blockquote>
Парный тег, указывает, что вложенный текст является расширенной цитатой.
fig
<figure></figure>
Парный тег, представляет собой автономное содержимое, потенциально с необязательным элементом подписи, который указывается с помощью <figcaption> элемента.
figcapt
<figcaption></figcaption>
Одиночный тег, представляет собой заголовок или легенду, описывающую остальное содержимое его родительского <figure> элемента
pic
<picture></picture>
Парный тег, предлагает альтернативные версии изображения для различных сценариев отображения / устройства.
cap
<caption></caption>
Парный тег, задает заголовок таблицы, предоставляя ей доступное описание.
colg
<colgroup></colgroup>
Парный тег, определяет группу столбцов в таблице.
fset
<fieldset></fieldset>
Парный тег, используется для группировки нескольких элементов управления.
btn
<button></button>
Парный тег, интерактивный элемент, активируемый пользователем с помощью мыши, клавиатуры, пальца, голосовой команды или другой вспомогательной технологии.
optg
<optgroup></optgroup>
Парный тег, создает группировку опций внутри <select> элемента.
